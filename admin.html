Add new order for your client
close

To add new order for an existing customer, select their name from drop-down list, then fill in rest of the form.
To add an order for a new Globex customer, follow the steps listed below:
1) Go to Sign up section, to create account for a new customer, using his email address and any generic password (for example "1234abcd").
2) After creating new customer account, you will be redirected to customers dashboard, where you can add new order.
3) Remember to give your customer their password and advise them to change it as soon as they log in for the first time.

close

To view order details, click "DETAILS" button.
To see which orders are currently waiting to be processed you can select "Awaiting orders" from "Sort orders" drop-down menu.
To change order status go to "Change status" column, then select status form drop-down menu.
All status changes will be available for your cutomers to view in their accounts.
After you have changed order status to "Processed" you can add tracking number in the relevant column.
